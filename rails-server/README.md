## Requirements

Ruby 3.3.6

## Setup

Install dependencies

- `bundle install`

Setup the database

- `rails db:create`
- `rails db:migrate`
- `rails db:seed`

## Running the app

- `rails s`

## About

Backend for an example accounting application.

Technology:

- Ruby on Rails: standard web application framework for Ruby
- Sqlite: light-weight relational database
